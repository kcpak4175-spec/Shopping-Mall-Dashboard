# TDD (Test-Driven Development) 기능 개발 워크플로우

이 워크플로우는 AI 어시스턴트가 프로젝트 내의 `tdd.md`에 명시된 TDD 원칙을 준수하여 기능을 구현하도록 합니다.

## 스텝
1. 사용자가 구현할 기능의 요구사항을 프롬프트 형태로 제공했는지 확인합니다.
2. `c:/Users/asia/Desktop/mall-dashboard/tdd.md` 파일의 [필수 준수 규칙] 섹션을 스스로 읽고 숙지합니다.
3. 숙지한 규칙(Red-Green-Refactor, 테스트 종합 작성, 커버리지 80% 이상)에 따라 순차적으로 개발을 진행하고 결과를 사용자에게 제시합니다.
4. 개발이 완료된 후, 모든 단위/통합/E2E 테스트가 정상적으로 통과하는지 최종적으로 검증하는 커맨드를 실행합니다.
